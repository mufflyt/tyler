# Mark a step as started

Mark a step as started

## Usage

``` r
mysterycall_progress_start(tracker, step, note = NULL)
```

## Arguments

- tracker:

  Object created by
  [`mysterycall_progress_tracker()`](https://mufflyt.github.io/mysterycall/reference/mysterycall_progress_tracker.md).

- step:

  Step name.

- note:

  Optional note stored alongside the step.

## Value

The tracker object, invisibly.

``` r

mysterycall_progress_start(tr, "Geocode")
#> [03:08:33] Started Geocode
#> [03:08:33] Progress: 0/2 steps complete (0.0%)
```
