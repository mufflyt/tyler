# Complete current step with timing

Complete current step with timing

## Usage

``` r
mysterycall_log_step_complete(
  success_rate = NULL,
  n_success = NULL,
  n_total = NULL
)
```

## Arguments

- success_rate:

  Optional success rate (0-1)

- n_success:

  Number of successful items

- n_total:

  Total items attempted

## Value

`invisible(NULL)` after printing duration and optional success metrics
for the current step.
