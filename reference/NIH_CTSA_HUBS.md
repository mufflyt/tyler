# NIH CTSA Hub Patterns

String patterns for NIH Clinical and Translational Science Award hub
institutions. All CTSA hubs are academic medical centers. Confidence:
0.99.

## Usage

``` r
NIH_CTSA_HUBS
```

## Format

Character vector of CTSA-related text patterns.
