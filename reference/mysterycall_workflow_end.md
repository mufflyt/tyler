# End workflow and print summary

End workflow and print summary

## Usage

``` r
mysterycall_workflow_end(final_n = NULL, input_n = NULL)
```

## Arguments

- final_n:

  Number of final output rows

- input_n:

  Number of input rows

## Value

Invisible NULL

``` r
if (FALSE) { # \dontrun{
```
