# Start a step in multi-progress tracker

Start a step in multi-progress tracker

## Usage

``` r
mysterycall_multi_step(tracker, step_num, total, detail = NULL)
```

## Arguments

- tracker:

  Multi-progress tracker object

- step_num:

  Step number (1-based)

- total:

  Total items in this step

- detail:

  Optional detail message

## Value

Invisible NULL
