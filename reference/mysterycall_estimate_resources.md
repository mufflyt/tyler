# Estimate workflow resources

Estimate workflow resources

## Usage

``` r
mysterycall_estimate_resources(n_rows)
```

## Arguments

- n_rows:

  Integer number of input rows used to project runtime and memory
  requirements.

## Value

List with runtime and memory estimates
