# ACGME OBGYN Residency Data

Information about Obstetricians and Gynecologists (OBGYN) residency
programs accredited by the Accreditation Council for Graduate Medical
Education (ACGME).

## Usage

``` r
acgme
```

## Format

A data frame with 142 columns:

- program_name:

  Name of the ACGME-accredited OBGYN residency program.

- address:

  Street address of the program.

- zip:

  ZIP code of the program.

- city:

  City where the program is located.

- state:

  State where the program is located.

- website:

  Program website URL.

- phone:

  Program contact phone number.

- program_code:

  ACGME program code.

- sponsoring_institution_code:

  ACGME code for the sponsoring institution.

- sponsoring_institution_name:

  Name of the sponsoring institution.

- original_accreditation_date:

  Date of original ACGME accreditation.

- accreditation_status:

  Current accreditation status.

- director_name:

  Name of the program director.

- director_date_appointed:

  Date the program director was appointed.

- coordinator_name_1:

  Name of the first program coordinator.

- coordinator_phone_1:

  Phone number of the first program coordinator.

- coordinator_email_1:

  Email address of the first program coordinator.

- participation_site_code_1:

  ACGME code for participation site 1.

- participation_site_name_1:

  Name of participation site 1.

- rotation_required_1:

  Whether rotation at site 1 is required.
