# Log success message

Log success message

## Usage

``` r
mysterycall_log_success(msg, details = NULL, indent = TRUE)
```

## Arguments

- msg:

  Message to log

- details:

  Optional list of details (name-value pairs)

- indent:

  Whether to indent (default TRUE)

## Value

`invisible(NULL)`.
