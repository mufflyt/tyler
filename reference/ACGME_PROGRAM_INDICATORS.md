# ACGME Program Indicators

String patterns indicating ACGME-accredited residency or fellowship
programs. Confidence: 0.98–0.99.

## Usage

``` r
ACGME_PROGRAM_INDICATORS
```

## Format

Character vector of ACGME-related text patterns.
