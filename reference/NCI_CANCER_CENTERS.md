# NCI-Designated Cancer Center Patterns

String patterns for National Cancer Institute designated cancer centers.
Confidence: 0.98.

## Usage

``` r
NCI_CANCER_CENTERS
```

## Format

Character vector of NCI cancer center text patterns.
