# Format duration in human-readable form

Format duration in human-readable form

## Usage

``` r
mysterycall_format_duration(seconds)
```

## Arguments

- seconds:

  Duration in seconds

## Value

Formatted string (e.g., "2h 34m 15s")

#> [1] "1h 1m 40s"
