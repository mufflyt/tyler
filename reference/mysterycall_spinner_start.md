# Show a spinner for indeterminate operations

Shows an animated spinner for operations where total is unknown.

## Usage

``` r
mysterycall_spinner_start(name, msg = NULL)
```

## Arguments

- name:

  Operation name

- msg:

  Optional message to display

## Value

Spinner ID (a `cli` progress bar ID, or `NULL` if cli is unavailable).
